# FactoCore-Figma-Project
FactoCore is a centralized dashboard concept built for multi-branch manufacturing companies to manage complex operations in one structured system.
Core modules include:
• Inventory management across multiple branches
• Finance and reporting dashboard
• Workforce and employee management
• Production tracking per branch
• AI-based stock prediction system


Design focus:
• Scalable dashboard architecture for enterprise use
• Clear information hierarchy for complex data systems
• Modular and reusable component system
• Multi-role and multi-branch UI structure
• Responsive layout for desktop and mobile use


Built entirely in Figma as an interactive prototype.
The prototype Link is given below from where You can check the responsiveness by clicking and testing the modules and different screens. Though it is not complete but it serves as a solid foundation for the complete version of FactoCore.


Prototype: https://www.figma.com/proto/I8b8ycwcoGhEMdrz6xkacV/FactoryCore?node-id=0-1&t=DiFPADRLCkTctMhp-1
